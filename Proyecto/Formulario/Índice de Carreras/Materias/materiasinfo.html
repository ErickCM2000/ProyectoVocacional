Informática

Tercer Semestre

  • Mantenimiento y Ensamblado de PC´s

    • En esta materia aprenderás las bases del hardware de una computadora. Conocerás los distintos componentes internos que tiene una computadora, sus funciones y su importancia.

  • Herramientas de Programación

    • Con esta materia aprenderás las bases de la programación. En principio usaras el software ClickCharts para crear diagramas de flujo y a partir del segundo parcial empezarás a programar en el lenguaje C++

Cuarto Semestre

  • Programación Estructurada

    • Esta materia será una continuación de la materia Herramientas de Programación, por lo que se verá un poco más en profundidad el lenguaje C++, y aprenderás a crear juegos básicos en este mismo lenguaje.

  • Realidad Virtual

    • Debido a limitaciones del equipo de cómputo de la escuela, no se utilizara software de realidad virtual. En su lugar, aprenderás a utilizar el lenguaje de programación Visual Basic y también a crear macros en Excel.

Quinto Semestre

  • Programación Orientada a Objetos

    • En esta materia se dejara a un lado el lenguaje C++ y aprenderás los fundamentos de la programación orientada a objetos (objetos, clases, métodos, parámetros).

  • Base de Datos

    • En esta materia aprenderás a generar conjuntos de datos e insertarlos en un software llamado Access. En esta materia podrás crear bases de datos, darles mantenimiento a las mismas.

  • Modelado de Sistemas

    • Aquí aprenderás a ver de forma detallada lo que hace tu programa. En esta materia ocuparas un software que se llama Visio, este mismo también es la base para realizar otro tipo de acciones o programas.

Sexto Semestre

  • Programación Avanzada

    • En esta materia obtendrás conocimientos sobre animaciones básicas o sobre aplicaciones desarrolladas para el sistema operativo Android. Para tus animaciones ocuparas un software que se llama Processing. Y para tus aplicaciones desarrolladas para Android ocuparas MII App Inventor 2.

  • Desarrollo Web y Multimedia

    • En esta materia aprenderás todo sobre el entorno de una página web. Empezarás a editar o hacer tus propias imágenes con todos los elementos de la suite de Adobe. También empezarás a hacer animaciones con flash. Tendrás que ser creativo y original a la hora de hacerla puesto a que al final tendrás que reunir todos tus conocimientos sobre HTML para poder realizar tu página web.

  • Teleinformática

    • Con esta materia aprenderás el comportamiento y funcionamiento de las redes, Aprenderás a realizar cables de red, cuantos metros te recomiendan para que tu red sea optima, podrás restringir el uso de tu internet etc. Esta materia se enfoca más en la práctica que en la teoría.

  • Sistemas de Información

    • En esta materia tendrás la oportunidad de escoger un proyecto para que lo estés realizando a lo largo del semestre, el proyecto que vayas a escoger tiene que ser relacionado a la carrera para así poder aplicar todos tus conocimientos adquiridos a lo largo de los semestres anteriores.
